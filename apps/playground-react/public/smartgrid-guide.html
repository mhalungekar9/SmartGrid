SmartGrid Guide

How to use the grid and its features

This page is a quick reference for the React SmartGrid implementation in this workspace. It covers the basic data setup, the major interactive features, and the keyboard/mouse behavior that the playground demo supports.

Sort Filter Resize Pin Edit Group Reorder Export CSV Pagination Column chooser

1. Minimal setup

Import the React wrapper, define a typed column list, pass your rows, and render the grid. Hidden, pinned, sortable, and editable behavior all come from the column config. 

import { SmartGrid, type Column } from "@smartgrid/react";

interface Employee {
  id: number;
  name: string;
  age: number;
  city: string;
  department: string;
  score: number;
}

const columns: Column<Employee>[] = [
  { id: "name", field: "name", headerName: "Name", width: 250, sortable: true, pinned: "left", editable: true },
  { id: "age", field: "age", headerName: "Age", width: 120, sortable: true, editable: true },
  { id: "city", field: "city", headerName: "City", width: 250, sortable: true, editable: true },
  { id: "department", field: "department", headerName: "Department", width: 220, sortable: true, editable: true },
  { id: "score", field: "score", headerName: "Score", width: 130, sortable: true, pinned: "right", editable: true },
];

<SmartGrid
  columns={columns}
  rows={rows}
  groupBy="department"
  pageSize={3}
/>

2. Core interaction model

The grid is designed around direct manipulation and spreadsheet-style navigation.

  • SortingClick a header to sort ascending or descending. Unsort by clicking again.
  • ResizingDrag the header edge to resize. Double-click to autosize to content.
  • Pinned columnsMark columns as pinned left or right to keep them anchored.
  • Quick filterSearch across visible columns using the toolbar input.

3. Editing and selection

Click a cell
Moves the active cell and selects the row.
Double-click a cell
Starts inline editing when the column is editable.
Enter or F2
Begins editing from the keyboard.
Escape
Cancels edit mode or clears the current selection anchor.
Arrow keys
Navigates between cells; hold Shift to extend range selection.

4. Clipboard behavior

Use Ctrl/Cmd+C to copy the active cell or selected range. Use Ctrl/Cmd+V to paste tab- separated text back into the grid. Numeric cells are parsed as numbers when possible.

Tip: You can paste a single value into a selected range and the value will fill the whole range.

5. Grouping, pagination, and reordering

  • Set groupBy to group rows by a field and collapse or expand groups.
  • Set pageSize to paginate the visible rows in the toolbar.
  • Drag a row by its handle to reorder rows within the current view.
  • Export the current visible rows to CSV from the toolbar.

6. Column chooser

Open the Columns button in the toolbar to hide or show fields at runtime. This uses the existing column metadata, so pinned state and widths remain part of the same config.

If a column is hidden in the chooser, it is removed from the visible layout, filter scope, and export surface for the current grid session.
